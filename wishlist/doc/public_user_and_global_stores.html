public_user_and_global_stores.js

example app: whiskie.net

whiskie is a music player frontend for tumblr.com. It allows you to make a favorites playlist that is publically visible. It also counts track plays to meassure its popularity

Models

  • Profiles
  • Tracks (have global play count)
  • Favorites (belong to track and profile)

Hoodie Challanges

  1. public profiles of users with their favorites
  2. global play counts of tracks

Scenario 2

I want to favorite or unfavorite a track

function favoriteTrack( track ) { hoodie.my.store.create( "favorite", { trackId = track.id }) } function unfavoriteTrack( track ) { hoodie.my.store.loadAll( "favorite", function(trackAtts) { return trackAtts.trackId === track.id }).done( function( arrTrackAtts ) { hoodie.my.store.delete( "favorite", arrTrackAtts[0].trackId ) }) }
